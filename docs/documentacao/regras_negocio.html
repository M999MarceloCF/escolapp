Regras de Negócio - EscolApp

Introdução

Este documento apresenta as regras de negócio que governam o funcionamento do EscolApp, um sistema simplificado de gestão escolar para dispositivos móveis, desenvolvido para atender às necessidades da Escola Municipal Professor Paulo Freire.

Regras Gerais

  1. RN001: O sistema deve operar em modo offline, permitindo a sincronização posterior quando houver conexão com a internet.
  2. RN002: Todas as informações sensíveis dos alunos devem ser protegidas e criptografadas.
  3. RN003: O acesso ao sistema deve ser controlado por perfis de usuário com diferentes níveis de permissão.

Perfis de Usuário

  1. RN004: O sistema deve suportar quatro perfis de usuário: Administrador, Professor, Aluno e Responsável.
  2. RN005: Apenas usuários com perfil Administrador podem cadastrar novos usuários no sistema.
  3. RN006: Professores só podem registrar notas e frequência para as turmas às quais estão vinculados.
  4. RN007: Alunos e Responsáveis só podem visualizar informações relacionadas ao próprio aluno ou aos alunos sob sua responsabilidade.

Registro de Notas e Frequência

  1. RN008: As notas devem ser registradas em uma escala de 0 a 10, com até uma casa decimal.
  2. RN009: A frequência deve ser registrada por aula, com as opções: Presente, Ausente ou Ausente Justificado.
  3. RN010: O sistema deve calcular automaticamente a média das notas e o percentual de frequência.
  4. RN011: O aluno será aprovado se obtiver média igual ou superior a 6,0 e frequência igual ou superior a 75%.

Comunicação

  1. RN012: Comunicados gerais podem ser enviados pela direção da escola para todos os usuários.
  2. RN013: Professores podem enviar comunicados específicos para alunos e responsáveis de suas turmas.
  3. RN014: Responsáveis podem enviar mensagens diretamente para professores e direção.
  4. RN015: Todos os comunicados devem ser armazenados localmente e sincronizados quando houver conexão.

Calendário Escolar

  1. RN016: O calendário escolar deve exibir eventos como feriados, reuniões, provas e atividades extracurriculares.
  2. RN017: Apenas a direção pode adicionar eventos ao calendário escolar geral.
  3. RN018: Professores podem adicionar eventos específicos para suas turmas.

Ocorrências Disciplinares

  1. RN019: Apenas professores e administradores podem registrar ocorrências disciplinares.
  2. RN020: Toda ocorrência disciplinar deve conter data, hora, descrição e medidas tomadas.
  3. RN021: Responsáveis devem ser notificados sobre ocorrências disciplinares de seus dependentes.

Sincronização de Dados

  1. RN022: A sincronização de dados deve ocorrer automaticamente quando o dispositivo estiver conectado à internet.
  2. RN023: Em caso de conflito de dados, prevalece a versão mais recente.
  3. RN024: O sistema deve manter um registro de todas as sincronizações realizadas.
