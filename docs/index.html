EscolApp

Sistema Simplificado de Gestão Escolar para Dispositivos Móveis

Sobre o Projeto

O EscolApp é um sistema simplificado de gestão escolar para dispositivos móveis, desenvolvido para atender às necessidades da Escola Municipal Professor Paulo Freire. O projeto foi concebido com foco em simplicidade, leveza e funcionamento offline, considerando as limitações de recursos de uma escola pública.

O sistema permite o registro de notas e frequência, comunicação entre escola e famílias, e acesso ao calendário escolar, mesmo em dispositivos mais antigos e com conexão de internet limitada.

Principais Características

  • Simplicidade: Interface intuitiva e fácil de usar
  • Leveza: Funciona em dispositivos com recursos limitados
  • Funcionamento Offline: Acesso às principais funcionalidades sem internet
  • Sincronização: Atualização de dados quando há conexão disponível
  • Multiplataforma: Desenvolvido em Python com Kivy/KivyMD

Telas Implementadas

O protótipo do EscolApp inclui as seguintes telas principais:

  • Login: Autenticação de usuários com diferentes perfis
  • Menu Principal: Hub central com acesso a todas as funcionalidades
  • Registro de Notas: Interface para professores registrarem notas dos alunos
  • Registro de Frequência: Controle de presença dos alunos
  • Comunicados: Sistema de mensagens entre escola e famílias
  • Calendário Escolar: Visualização de eventos e datas importantes
  • Ocorrências: Registro de ocorrências disciplinares

Todas as telas foram projetadas seguindo princípios de design centrado no usuário, com foco em simplicidade, usabilidade e acessibilidade.

Tecnologias Utilizadas

Python

Linguagem de programação principal, escolhida por sua simplicidade, legibilidade e ampla comunidade de suporte.

Kivy/KivyMD

Framework para desenvolvimento de aplicações móveis multiplataforma com Python, oferecendo componentes de interface modernos e responsivos.

SQLite

Banco de dados leve e embutido, ideal para aplicações móveis com funcionamento offline.

MVC

Padrão arquitetural Model-View-Controller para organização do código e separação de responsabilidades.

Conclusão

O EscolApp representa uma solução simples e eficaz para os desafios de gestão escolar em instituições públicas com recursos limitados. Seu design centrado no usuário, funcionamento offline e baixos requisitos de hardware o tornam acessível mesmo em contextos com infraestrutura tecnológica reduzida.

O projeto demonstra como a tecnologia pode ser adaptada para atender necessidades específicas, contribuindo para a melhoria da comunicação entre escola e famílias e para a otimização dos processos administrativos e pedagógicos.
